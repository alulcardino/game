Суть игры в следующем:
Ты путешественник и попал в пешеру.
У тебя нет другого выхода, кроме как идти вперед.
Перемещаемся с помощью W,S,D.
Как только достиг знака вопроса, с помощью рандомf определяем, что за событие случится.
(50% на встречу с монстром, 10% на математическую задачу, аптечку, яму, подарок и выгодное предложение)
У игрока есть 7 характеристик(ум, броня, правая рука, левая рука, ноги, уровень и здоровье)
Игра закончиться в тот момент, как показатели жизни или уровня будут ровны 0.
Большинство геймплейных вещей определяются с помощью рандома. То есть урон монстра, качество награды и так далее.
Было реализовано 3 класса: Player, Enemy и Event. Для каждого есть отдельный файл .h и .cpp. Вся реализация в cpp;
Помимо геймплея есть таблица рекордов. Для ее создания была подключениа зависимость <vector>, с помощью которого можно было динамически добавлять 
результаты игрока. Сама же таблица находится в файле records.txt в корневом каталоге проекта. Был использован класс исключений для проверки существования файла.
Помимо этого так же есть 2 файла interface и logic для функций, которые размещать в main было нецелесообразно.
Были использованы зависимости <conio> и <dos>, следовательно проект будет работать только на Windows. На unix-подобных программах не были найдены их аналоги, 
Так что пришлось писать на Windows (Также весомым аргументом в пользу Windows была Visual Studio).
Надеюсь, этого окажется достаточным для вступления в вашу команду:)
